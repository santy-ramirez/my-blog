Paginas web para aprender html

April 17, 2022 · One min read
Santiago Ramirez
web developer

Para dedicarte al desarrollo web sin duda unas de las primeros recursos que uno tiene que aprender es html, que es el encargado de darle estructura a nuestras paginas web,¿ Que es html?

HTML sus siglas en inglés de HyperText Markup Language (lenguaje de marcado de hipertexto), hace referencia al lenguaje de marcado  para la elaboración de páginas web , es el encargado de presentar los datos en nuestra web es a continuacion dejo algunas paginas donde podemos podemos profundizar en el aprendizaje conocer los diferentes etiquetas y propiedades:

  • 1 w3school: Es un sitio web para aprender tecnologías web . Contiene tutoriales ademas de HTML, de CSS, JavaScript, SQL, PHP ymucho mas.

  • 2 MDN Web Docs: es el sitio web oficial de Mozilla para la documentación de estándares web aloja muchos documentos sobre diferentestes tecnologías web,como HTML5, JavaScript, CSS, y mucho mas, exelente si quieres profundizar en html.

  • 3 HTMLreference: Cheetseet de tags de html, exelente para consultar cuando olvidas como usar los tags, te muestra una lista con ejemplos de como implimentarlos.
